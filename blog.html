Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML use to insert dynamic HTML content. 2. innerText use to get only visible text. 3. textContent use to get all text, including hidden.

Question-3: What is event delegation in the DOM?

Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM hierarchy, allowing parent elements to respond to events triggered by their child elements. This means the event starts at the target element and propagates upward through its parent elements to the root of the DOM.
